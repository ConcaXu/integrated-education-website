# Integrated Education & Business Group (WanderChina.Guide) Website

## 项目简介
这是 Integrated Education & Business Group (WanderChina.Guide) 的官方网站项目。该网站旨在为全球用户提供中国研学旅行、政企出海考察及文化旅游的一站式服务信息。

## 快速开始
本项目为静态网站，无需复杂的后端环境配置。

1.  **克隆或下载** 项目代码到本地。
2.  直接双击打开 `index.html` 即可在浏览器中预览。
3.  建议使用 VS Code 等编辑器配合 Live Server 插件进行开发预览。

## 项目结构
*   `index.html`: 首页
*   `products.html`: 产品目录页
*   `product-detail.html`: 产品详情页 (动态渲染)
*   `activities.html`: 近期活动列表页
*   `activity-detail.html`: 活动详情页 (动态渲染)
*   `about.html`: 关于我们
*   `css/`: 样式文件
*   `js/`: 脚本及数据文件
*   `assets/`: 图片及视频资源
